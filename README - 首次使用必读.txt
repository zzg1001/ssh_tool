                    SSH Tool 首次使用说明
═══════════════════════════════════════════════════════════════

感谢您使用 SSH Tool！

由于 macOS 的安全机制，未经 Apple 签名的应用在首次运行时
会被系统阻止。这是正常现象，请按以下步骤操作：


【推荐】方法一：运行安装脚本
───────────────────────────────────────────────────────────────
1. 双击「安装说明.command」文件
2. 如果提示"无法打开"，请右键点击它，选择「打开」
3. 按照屏幕提示操作即可


方法二：手动设置
───────────────────────────────────────────────────────────────
1. 右键点击 SSH Tool.app（或按住 Control 键点击）
2. 在菜单中选择「打开」
3. 在弹出的对话框中，点击「打开」按钮
4. 首次打开后，以后就可以正常双击打开了


方法三：通过系统设置
───────────────────────────────────────────────────────────────
1. 尝试双击打开 SSH Tool.app（会被阻止）
2. 打开「系统设置」→「隐私与安全性」
3. 向下滚动，找到关于 SSH Tool 被阻止的提示
4. 点击「仍要打开」


【高级】方法四：终端命令
───────────────────────────────────────────────────────────────
打开「终端」应用，输入以下命令：

    xattr -cr "/Applications/SSH Tool.app"

（如果应用在其他位置，请替换路径）


═══════════════════════════════════════════════════════════════
                         常见问题
═══════════════════════════════════════════════════════════════

Q: 为什么会出现安全提示？
A: macOS 会阻止未经 Apple 签名的应用运行，这是系统的
   安全保护机制。本应用是安全的，只是未购买 Apple 的
   开发者签名证书。

Q: 这样操作安全吗？
A: 是的。您只是告诉系统"我信任这个应用"，不会影响
   系统对其他应用的安全检查。

Q: 每次打开都需要这样做吗？
A: 不需要。只有首次打开需要手动确认，之后可正常使用。
